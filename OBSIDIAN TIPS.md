# Callout Blocks
- ```
    > [!NOTE] Hardware Configuration
    > The on-prem server is a Proxmox hpe proliant ml150 gen9.
    ```
    

**Common Callout Types (`TYPE`):**

|**Type**|**Use Case**|
|---|---|
|`NOTE`|General information or facts.|
|`INFO`|Highlights a useful tip or resource.|
|`TIP`|General advice (good for cybersecurity).|
|`WARNING`|Something important to be careful about (e.g., _don't run a command_).|
|`QUESTION`|A block for self-reflection or a prompt.|
