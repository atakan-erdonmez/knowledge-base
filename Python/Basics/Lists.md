```python
tr_sehirler = ['adana', 'ankara', 'istanbul']

print(tr_sehirler[0])

tr_sehirler[1] = 'balikesir'
```

When you create a list like this, the order is preserved.

# Nested Lists
```python
list1 = ['a', 'b', 'c']
list2 = ['z', 'y', 'x']

nested_list = [list1, list2]
```

When you print `nested_list[0]`, it will print all list1
When you print `nested_list[0][1]`, it will print 'b'
