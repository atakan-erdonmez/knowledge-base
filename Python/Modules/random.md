# Main Functions

### random.random()
0.0 <= N < 1.0

Used to create random floats

### random.randint(a, b)
a <= N <= b

Creates integer

### random.uniform(a, b)
Creates a *float* between a and b

## random.choice(list)
Choose a random item from a given list

## random.shuffle(list)
Shuffle the list
